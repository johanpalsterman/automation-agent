Wat is dit systeem?
WishFlow AI bestaat uit twee lagen die samenwerken om een slimme, autonome AI-assistent te vormen.
🦞
OpenClaw — de Agent
De AI-agent — de werknemer. Verwerkt berichten, denkt na, voert acties uit. Heeft toegang tot tools, geheugen en extern internet. Verbindt via WebSocket.
📎
Paperclip — de Orchestrator
De orchestratielaag — het bedrijf. Beheert doelen, taken en meerdere agents. Geeft structuur aan complexe workflows. Optioneel maar krachtig.
🌉
De Gateway — de Brug
De brug tussen jou en de agent. Beheert authenticatie via het token, WebSocket-verbindingen, en routeert berichten. Draait op poort 5000.
Hoe werkt het starten?
Het systeem start automatisch via start.sh. Geen handmatige installatie nodig.
AUTO
start.sh installeert automatisch alle software (openclaw + paperclipai via npm) als die nog niet aanwezig is.
SLIM
Software wordt vers geinstalleerd bij elke run als die ontbreekt. Verwijder handmatig en herstart om updates te forceren.
VEILIG
Werkt op terminal servers met minimale rechten. Geen sudo nodig. Alles installeert in de home-directory via npm global install.
ACHTERGROND
Alle processen (gateway, agent, Paperclip) starten op de achtergrond. Logs zijn beschikbaar als bestanden in de werkdirectory.
Als de Replit sessie verloopt en je de Run-knop indrukt, herstart alles automatisch. Geen verdere actie nodig.
Verbinding maken — stap voor stap
Volg deze stappen om verbinding te maken met de AI-agent via de Control UI.

  1. Klik op de knop OpenClaw Control UI openen op het Dashboard tabblad. De Control UI opent in een nieuw venster.

  2. Je ziet een verbindingsscherm. Kopieer de WebSocket URL van het Dashboard (begint met wss://) en plak die in het veld Gateway URL.

  3. Voer het token in: openclaw-replit-secret (of de waarde van de OPENCLAW_TOKEN secret als je die hebt aangepast).

  4. Klik op Connect. De verbinding wordt tot stand gebracht en je ziet een bevestiging.

  5. Je bent nu verbonden! Je kunt chatten met de AI-agent, taken geven, bestanden laten analyseren en meer.

Zorg dat de gateway actief is (groene pulserende dot op het Dashboard). Als die er niet is, herstart dan via de Replit Run-knop.

Messaging koppelen
Verbind de AI-agent met Telegram of Discord om overal mee te chatten.
Telegram koppelen

  1. Open Telegram en zoek @BotFather

  2. Stuur /newbot en volg de instructies

  3. Kopieer het token dat je ontvangt

  4. Voeg toe als Replit Secret: TELEGRAM_BOT_TOKEN

  5. Herstart via de Run-knop. De bot is actief!

Discord koppelen

  1. Ga naar discord.com/developers

  2. Maak een nieuwe Application aan

  3. Ga naar Bot en kopieer het token

  4. Voeg toe als Replit Secret: DISCORD_BOT_TOKEN

  5. Herstart via de Run-knop en invite de bot

Veelgestelde vragen en Problemen oplossen
Cannot connect of verbinding mislukt
Controleer of de groene dot actief is. Zo niet, herstart via de Replit Run-knop. Controleer de WebSocket URL (begint met wss://) en het token (openclaw-replit-secret).
Internal server error of agent reageert niet
Controleer of LLM_API_KEY correct is ingesteld. Bij LLM_PROVIDER=replit is geen API-sleutel nodig. Bij andere providers is een geldige API-sleutel vereist.
Model not found of onbekend model
Controleer of LLM_PROVIDER overeenkomt met het type API-sleutel. Gebruik niet een OpenAI-sleutel met LLM_PROVIDER=groq. Gebruik replit voor de ingebouwde Replit AI.
Terminal server zonder admin-rechten
start.sh werkt volledig zonder sudo. Alle software installeert in de home-directory via npm global install. Geen root-rechten nodig.
Replit sessie verloopt (pagina wordt wit)
Dit is normaal. Druk op de Run-knop in Replit om alles te herstarten. Ververs dan deze dashboardpagina. Alle secrets blijven bewaard.
Paperclip Dashboard laadt niet
Paperclip draait op poort 3001. Wacht 10-20 seconden na het starten. Paperclip start iets later dan de gateway. Herlaad de pagina.

Voorbeeldcommandos

export LLM_PROVIDER=openai
export LLM_API_KEY=sk-your-key-here

export LLM_PROVIDER=groq
export LLM_API_KEY=gsk_your-key-here

export OPENCLAW_TOKEN=mijn-eigen-veilig-token

export TELEGRAM_BOT_TOKEN=123456789:ABCdef...
Discord bot instellen
export DISCORD_BOT_TOKEN=your-discord-token
